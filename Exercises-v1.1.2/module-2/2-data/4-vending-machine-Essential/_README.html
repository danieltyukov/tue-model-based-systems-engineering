4. Vending machine EssentialÔÉÅ

Model a vending machine that dispenses drinks costing 25 eurocents. Assume that depositing can be done in quantities of 5, 10 or 20 eurocents. Let us name the event of depositing these d5, d10 and d20 respectively. It is possible to ask for a drink using event D and for a return of the coins using the event R. Drinks will only be dispensed when enough money has been deposited.
